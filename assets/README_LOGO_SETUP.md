## To Display Logo on GitHub

1. **Add the logo to Git:**
   ```bash
   git add assets/logo.png
   git commit -m "Add: Project logo for README"
   ```

2. **Push to GitHub:**
   ```bash
   git push origin main
   ```

3. **Verify on GitHub:**
   - Visit: https://github.com/karthikredddy7github/Flood-Forecasting-Model-Using-Federated-Learning
   - The logo should now display in the README

### Option 2: Create Your Own Logo
If you want to customize the logo:
1. Create a PNG image (recommended size: 300-600px width)
2. Save it as `assets/logo.png`
3. Commit and push to GitHub
